# IRON HOME — Витрина спортивного инвентаря

Прототип сайта для продажи спортивного инвентаря (турники, брусья, скамьи, стойки).
Красно-белый премиальный дизайн, каталог товаров, админ-панель.

## Быстрый запуск

1. Откройте папку `sport-vitrina`
2. Дважды кликните на **`index.html`**
3. Сайт откроется в браузере

> Для фото товаров нужен интернет (изображения с Unsplash).
> Админка работает офлайн — данные хранятся в браузере (localStorage).

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Главная — hero, категории, хиты продаж |
| `catalog.html` | Каталог с фильтрами по категориям |
| `product.html` | Карточка товара с кнопками Kaspi и WhatsApp |
| `admin.html` | Админ-панель управления товарами |

## Админ-панель

- URL: откройте `admin.html`
- **Пароль по умолчанию:** `admin123`
- Можно: добавлять/редактировать/удалять товары, менять настройки (телефон, WhatsApp, название компании)

## Что настроить под себя

В админке → **Настройки**:
- Название компании
- Телефон и WhatsApp
- Ссылки на Kaspi магазин
- Пароль админки

Для каждого товара укажите:
- Реальную ссылку на карточку в Kaspi
- Фото (URL или загрузите на хостинг позже)

## Дальнейшее развитие

Это прототип для просмотра. Для продакшена рекомендуется:
- Перенести на **Next.js** или **WordPress** с базой данных
- Подключить реальный хостинг и домен
- Заменить localStorage на серверное хранение
- Добавить загрузку фото через админку
- Подключить аналитику (Google Analytics / Yandex Metrika)
