# SPORT KING — Design System

## Brand positioning
Premium home fitness brand. Clean, strong, modern, minimal, industrial, sporty.

**Tagline:** ТРЕНИРУЙСЯ ДОМА. СТАНОВИСЬ СИЛЬНЕЕ.

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary` | `#D9FF00` | Buttons, active nav, badges, accents |
| `--color-primary-hover` | `#C5EB00` | Button hover |
| `--color-background` | `#F7F7F4` | Page background |
| `--color-surface` | `#FFFFFF` | Cards, panels |
| `--color-dark` | `#111313` | Headings, dark sections |
| `--color-dark-secondary` | `#191C1D` | Footer, dark UI |
| `--color-text` | `#111313` | Body text |
| `--color-text-secondary` | `#6D706D` | Secondary text |
| `--color-border` | `#E5E5E0` | Borders |
| `--color-success` | `#3C9B5F` | In stock |

**Rule:** Lime `#D9FF00` is accent only — never as large background.

## Typography
- **Primary font:** Manrope (fallback: Inter, system-ui)
- **H1:** 72–96px desktop, weight 800
- **H2:** 42–56px, weight 800
- **H3:** 20–28px, weight 700
- **Body:** 16–18px
- **Small:** 12–14px

## Spacing & radius
- Section padding: `py-16 md:py-24`
- Container: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card radius: `rounded-2xl`
- Button radius: `rounded-xl`
- Button min-height: `48px`

## Components
- **Primary button:** lime bg, dark text, hover scale 1.02
- **Secondary button:** dark bg, white text
- **Outline button:** border, transparent bg
- **Product card:** white surface, subtle border, hover scale 1.02
- **Kaspi CTA:** always «КУПИТЬ НА KASPI»

## Animations (allowed only)
- Fade in, slide up
- Hover scale ≤ 1.02
- Button hover transitions
- Smooth navigation

## Forbidden
Grid/checker backgrounds, cyberpunk, random gradients, glow, glassmorphism, emoji UI, neon, parallax, spinning elements.
